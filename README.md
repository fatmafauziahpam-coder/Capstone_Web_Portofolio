# 🌸 Personal Portfolio Website - Flask

Website portfolio pribadi yang dibangun menggunakan **Python Flask**, **Flask-SQLAlchemy**, **Flask-Login**, dan **SQLite** sebagai database.

---

## 📌 Fitur

- Login Admin
- Dashboard Admin
- CRUD Project
- CRUD Profile
- Contact Form
- Menampilkan pesan dari pengunjung
- SQLite Database
- Responsive Portfolio

---

## 🛠 Teknologi

- Python
- Flask
- Flask-SQLAlchemy
- Flask-Login
- SQLite
- HTML5
- CSS3
- JavaScript

## 💻 Cara Menjalankan

### Clone Repository

```bash
git clone https://github.com/fatmafauziahpam-coder/Capstone_Web_Portofolio.git
```

Masuk ke folder project

```bash
cd Capstone_Web_Portofolio
```

Install library

```bash
pip install -r requirements.txt
```

Jalankan project

```bash
python app.py
```

Buka browser

```
http://127.0.0.1:5000
```
